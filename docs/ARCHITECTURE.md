# Архитектура проекта Акселион (Python калькулятор)

## Обзор
Проект представляет собой консольный калькулятор на Python с поддержкой базовых арифметических операций, историей вычислений и обработкой ошибок.

## Структура проекта
\`
project/
├── src/
│   ├── calculator.py      # Основной модуль калькулятора
│   ├── operations.py      # Арифметические операции
│   └── history.py         # Управление историей вычислений
├── tests/
│   ├── test_calculator.py # Тесты для калькулятора
│   └── test_operations.py # Тесты для операций
├── docs/                  # Документация
├── README.md              # Основное описание проекта
└── requirements.txt       # Зависимости проекта
\`

## Модули

### calculator.py
Основной модуль приложения. Содержит:
- Класс Calculator - основной класс калькулятора
- Логику взаимодействия с пользователем
- Обработку ввода/вывода

### operations.py
Модуль арифметических операций:
- Сложение, вычитание, умножение, деление
- Возведение в степень
- Остаток от деления

### history.py
Модуль управления историей вычислений:
- Сохранение операций
- Отображение истории
- Очистка истории

## Взаимодействие модулей
Пользователь → calculator.py → operations.py/history.py → Результат

## Обработка ошибок
- Проверка деления на ноль
- Проверка корректности ввода
- Логирование ошибок
