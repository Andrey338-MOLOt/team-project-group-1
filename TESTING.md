# План и результаты тестирования проекта Акселион

## Цели тестирования
1. Проверка корректности арифметических операций
2. Проверка работы системы истории вычислений
3. Проверка обработки ошибок
4. Проверка пользовательского интерфейса

## Методы тестирования

### 1. Модульное тестирование
- Использование pytest для автоматических тестов
- Тестирование каждой функции калькулятора отдельно
- Проверка граничных случаев

### 2. Интеграционное тестирование
- Проверка взаимодействия модулей
- Тестирование полного цикла операций

### 3. Ручное тестирование
- Проверка пользовательского интерфейса
- Тестирование ввода неверных данных
